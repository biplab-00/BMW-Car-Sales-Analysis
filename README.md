# BMW-Car-Sales-Analysis

 Project Overview
This project analyses BMW car sales data from 2010 to 2024, covering 11 different BMW models. The objective is to uncover meaningful insights into sales performance, revenue trends, regional distribution, and feature-level impact to support data-driven business decisions. The project demonstrates an end-to-end analytics workflow—from data cleaning and transformation to visualization and insight generation.

 Objectives
•	Analyse sales and revenue growth trends over time 
•	Identify top-performing regions and markets 
•	Evaluate model-wise performance and contribution 
•	Understand the impact of features (fuel type, mileage, transmission, engine size) on sales 
•	Deliver actionable insights through interactive dashboards 

Tools & Technologies
•	Python – Data cleaning, preprocessing, and analysis 
•	Microsoft Power BI – Dashboard creation and data visualization 


 Key Insights
•	 Strong post-2020 recovery with consistent growth in recent years.
•	 Asia leads sales, followed by Europe and North America.
•	 Balanced contribution across fuel types, with slight dominance of hybrid and petrol.
•	 Automatic transmission vehicles outperform manual models.
•	 Mid-to-high engine capacity vehicles dominate sales. 
•	 Sales show temporary dips during external disruptions, but maintain long-term stability.
